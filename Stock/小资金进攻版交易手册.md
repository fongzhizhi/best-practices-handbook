# 小资金进攻版交易手册
> **适用阶段**：本金 8.5 万 → 30 万（目标两年内完成）  
> **核心理念**：在严格风控前提下，适度集中、放大弹性，用 4-5 个中线波段实现账户快速积累。  
> **与稳健版差异**：放宽估值容忍度，提高首次仓位，增加强催化权重，优化止盈结构，提升交易频率。  
> **CreateDate**: 2026-04-02  
> **UpdateDate**: 2026-04-02  

---

## 一、选题材标准（从宏观到产业）

**核心原则：选择供给端刚性锁定 + 需求端稳定增长 + 短期内有强催化预期的行业。**  
**强催化**：未来 3-6 个月有明确的、可预见的利好事件（产品调价、政策落地、旺季备货、海外订单、技术突破等）。

用“四看”筛选：

| 维度       | 标准                                                         | 三美案例                          |
| ---------- | ------------------------------------------------------------ | --------------------------------- |
| **看供给** | 政策、配额、技术壁垒导致供给无法随意增加                     | 三代制冷剂配额锁死                |
| **看需求** | 下游需求刚性，且近期有新增长点（出口/替代）                  | 空调+汽车保有量+冷链+海外补库     |
| **看价格** | 核心产品价格处于上涨通道或高位稳定，且近期有上调预期         | R32 二季度长协价上涨 8%           |
| **看催化** | 未来 3-6 个月有明确事件驱动（财报季、产品涨价、政策、行业会议） | 4 月空调旺季 + 印度中东配额收官年 |

**操作**：每周花 15 分钟追踪 2-3 个潜在板块的核心变量（价格、库存、开工率、政策动态），找到“价格上行 + 催化临近”的交集。

## 二、选股标准（从行业到公司）

**核心原则：在景气行业中，选择市占率领先、业绩弹性大、估值相对合理、业务聚焦的公司。**  
**适度放宽 PE/PEG 容忍度，但必须满足“可跟踪”和“逻辑清晰”。**

用“五层筛子”：

| 筛子         | 标准                                             | 三美案例                             |
| ------------ | ------------------------------------------------ | ------------------------------------ |
| **行业地位** | 市占率前三或细分龙头，拥有定价权                 | 三代制冷剂配额占比约 18%，R134a 领先 |
| **业绩弹性** | 最近财报净利润增速 > 50%，且涨价能直接转化为利润 | 2025 前三季度 +183.66%               |
| **估值容忍** | 动态 PE < 30 倍，PEG < 1（原 <20、<0.5）         | 动态 PE 约 18 倍，PEG 0.09           |
| **业务聚焦** | 主营产品占营收比例高，涨价弹性大                 | 制冷剂营收占比约 85%                 |
| **可跟踪性** | 只需跟踪 1-2 个高频数据（价格、库存）            | 每周查 R32/R134a 价格                |

**操作**：用财务软件筛选 PE<30、Q3 增速 > 50% 的股票，再从中挑行业龙头 + 有明确催化。

## 三、买入标准（进攻型布局）

**核心原则：根据确定性高低决定首次仓位，左侧右侧并行，分批建仓但敢于在强催化前加大仓位。**

### 1. 左侧买入（下跌中布局）

| 条件     | 标准                                                         | 三美案例                                                     |
| -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 逻辑未变 | 产品价格没跌、公司没利空、行业政策未转向                     | 制冷剂价格仍处高位                                           |
| 跌幅到位 | 从近期高点回撤 15%-25%                                       | 从 76 元回撤至 60 元（-21%）                                 |
| 估值便宜 | 动态 PE 进入历史低位区域或明显低于行业平均                   | PE 从 20 倍压缩至 18 倍                                      |
| 技术支撑 | 靠近 60 日线或前期成交密集区                                 | 60.8 元接近 60 日线                                          |
| 仓位计划 | **确定性高（强催化明确）首次可 30%-40%，中等确定性首次 20%** | 300 股（21%）为中等确定性，若当时已知二季度长协价大涨预期，可提至 35% |

### 2. 右侧买入（趋势确认后）

| 条件     | 标准                                               |
| -------- | -------------------------------------------------- |
| 突破阻力 | 放量突破近期平台或重要均线（如 60 日线）           |
| 回踩确认 | 突破后缩量回踩不破支撑                             |
| 催化临近 | 强催化事件即将发生（如产品涨价公告、旺季备货启动） |
| 仓位计划 | **首次 20%-30%**                                   |

**操作**：提前写好计划——买入价格区间、数量、止损线、止盈线。到价挂单，不临时决策。  
**小资金特别提示**：当逻辑高度确定且催化迫近时，可以一次性建立 30%-40% 仓位，剩余仓位留作加仓或机动。

## 四、卖出/止盈标准（让主仓位奔跑）

**核心原则：不贪最后一个铜板，但要让大部分仓位吃到主升浪。**  
**止盈结构调整**：第一、第二止盈各卖 20%，剩余 60% 博更高收益。

| 批次         | 止盈条件                                     | 卖出比例           | 三美案例（成本 60.8）         |
| ------------ | -------------------------------------------- | ------------------ | ----------------------------- |
| **第一止盈** | 达到成本 +10%~15%                            | 卖出持仓的 **20%** | 68-70 元卖 60 股（原 100 股） |
| **第二止盈** | 达到成本 +20%~30%                            | 再卖持仓的 **20%** | 75-78 元再卖 60 股            |
| **剩余仓位** | 逻辑未变则继续持有，可设置移动止损或跟踪止盈 | **60%** 博更高     | 剩余 180 股看 80 元以上       |

**特殊止盈**：若行业出现重大利空（如产品价格连续两周下跌、政策逆转），立即清仓。

**移动止损替代方案**：剩余仓位可采用“跌破 20 日线或 10 日线减半”等动态止盈方式，保护利润同时保留上行空间。

## 五、止损标准（保护本金，永不改变）

**核心原则：宁可卖错，不可被深套。止损位一旦设定，盘中不修改。**

| 类型         | 标准                                         | 三美案例                           |
| ------------ | -------------------------------------------- | ---------------------------------- |
| **固定止损** | 买入价的 -8% ~ -10%                          | 60.8 元买入，止损设 56 元（-7.9%） |
| **移动止损** | 股价上涨后，将止损上移至成本价或关键支撑     | 若涨至 70 元，止损可上移至 65 元   |
| **逻辑止损** | 核心逻辑被证伪（产品价格趋势下跌、业绩变脸） | 制冷剂价格连续下跌 10% 则止损      |

**操作**：买入后立即设好条件单，跌破止损价自动卖出。不抱幻想，不补仓（除非是计划内的加仓）。

## 六、仓位管理（集中但不孤注一掷）

**核心原则：小资金集中火力，但永远不一次打满，用分批建仓控制风险。**

| 资金规模   | 单只股票最大仓位 | 首次建仓比例        | 加仓后最大比例 |
| ---------- | ---------------- | ------------------- | -------------- |
| 8.5 万阶段 | **70%**          | 20%-40%（视确定性） | 70%            |

**加仓条件**：
- 必须等价格回落到预设区间，或右侧确认信号出现
- 加仓后总仓位不超过上限
- 逻辑未变，且股价未触发止损

**三美案例**：总资金 8.5 万，首次买入 300 股（1.82 万，占 21%）。若确定性提高，可首次买入 500 股（约 3 万，占 35%）。计划若跌至 58-60 元再加 200 股（累计约 4.5 万，占 53%），仍在安全范围内。

## 七、交易频率与节奏

**核心原则：一年做 4-5 个中线波段，空仓期不超过 2 个月。**

| 阶段     | 任务                               |
| -------- | ---------------------------------- |
| 1-2 月   | 布局年报预增 + 春季旺季品种        |
| 3-4 月   | 主做一季报 + 夏季旺季（如制冷剂）  |
| 5-6 月   | 逐步兑现，布局中报预期             |
| 7-8 月   | 夏季题材（旅游、电力）或中期业绩浪 |
| 9-10 月  | 秋季旺季 + 三季报                  |
| 11-12 月 | 跨年行情 + 下一年度预期            |

**操作**：每完成一个波段，用 1-2 周复盘、筛选下一个板块，保持节奏。

## 八、跟踪与复盘（保持敏锐）

### 日常跟踪（每日收盘 10 分钟）

| 项目     | 内容                                  | 工具            |
| -------- | ------------------------------------- | --------------- |
| 收盘价   | 记录持仓股收盘价，确认未触发止损/止盈 | 行情软件        |

| 资金流向 | 查看主力资金、融资余额变化            | 同花顺/东方财富 |

### 每周跟踪（周日晚上 30 分钟）

| 项目       | 内容                                                   | 工具                       |
| ---------- | ------------------------------------------------------ | -------------------------- |
| 核心变量   | 查产品价格、库存、开工率                               | 生意社、隆众资讯、百川盈孚 |
| 行业新闻   | 扫一眼行业重大政策、公司公告、海外动态                 | 财联社、证券时报           |
| 催化日历   | 列出未来一个月行业重要事件（财报、会议、价格调整窗口） | 自制日历                   |
| 同板块对比 | 看龙头股走势，确认板块趋势                             | 行情软件                   |

| 项目     | 内容                                                     |
| -------- | -------------------------------------------------------- |

| 目标检查 | 对照两年 30 万目标，看进度是否符合预期                   |

## 九、心态与纪律（进攻版特别强调）

1. **只看收盘价，盘中不决策**：90% 波动是噪音，只有收盘价有意义。  
2. **计划内重仓，计划外空仓**：高确定性时敢于重仓（40%），但绝不做临时单。  
3. **不预测顶底，分批操作**：用分批买入/卖出应对不确定。  
4. **不频繁换股**：一年 4-5 个波段，每个波段 1-2 只核心标的，反复跟踪验证。  
5. **不比较收益**：别人的涨停与你无关，只关注自己的曲线。  
6. **接受波动**：小资金进攻必然伴随更大回撤（15% 以内正常），只要逻辑未变，不因短期浮亏而动摇。  
7. **牢记目标**：8.5 万到 30 万是两年任务，不是一个月，耐心比爆发力更重要。

## 十、小资金进阶路线图

### 阶段一：8.5 万 → 15 万（第一年目标）

| 时间      | 任务                                            | 关键点                     |
| --------- | ----------------------------------------------- | -------------------------- |
| 前 3 个月 | 验证策略，做 1-2 个波段，确保执行纪律           | 胜率不重要，关键是按计划做 |
| 中 6 个月 | 逐步提高确定性判断，在强催化时敢于 30%-40% 仓位 | 选对行业，重仓吃主升       |
| 后 3 个月 | 累积收益，账户突破 15 万                        | 复盘修正，提升频率         |

### 阶段二：15 万 → 30 万（第二年目标）

| 时间   | 任务                                          | 关键点                   |
| ------ | --------------------------------------------- | ------------------------ |
| 上半年 | 继续保持每年 4-5 个波段节奏，单票仓位可至 70% | 利用复利效应加速         |
| 下半年 | 账户突破 30 万后，准备转向稳健版              | 逐步降低进攻性，保护本金 |

### 阶段三：30 万以上（转为稳健版）

达到 30 万后，切换回原《“五位一体”交易手册》稳健模式，单票仓位降至 30%-40%，追求年化 20%-30% 的可持续复利，向 100 万迈进。

## 附：三美股份实战推演（进攻版应用）

| 阶段     | 稳健版动作                                | 进攻版动作（优化）                        | 效果差异           |
| -------- | ----------------------------------------- | ----------------------------------------- | ------------------ |
| 选题材   | 供给刚性+需求增长+价格上行                | **+二季度长协价上调预期（强催化）**       | 提前加大信心       |
| 选股     | PE<20，业绩 >50%                          | PE<30，业绩 >50%                          | 放宽筛选范围       |
| 买入     | 首次 20% 仓位（300 股）                   | 若已知催化，首次可 **35% 仓位（500 股）** | 初始仓位翻倍       |
| 加仓     | 预设 58-60 元加 200 股                    | **若右侧突破 63 元，可追加 100 股**       | 多一次加仓机会     |
| 止盈     | 第一 30%，第二 30%，剩余 40%              | **第一 20%，第二 20%，剩余 60%**          | 主仓位更多吃主升   |
| 理论结果 | 若涨至 80 元，总盈利约 1.2 万（14% 收益） | 若涨至 80 元，总盈利约 2.2 万（26% 收益） | 同样行情，收益翻倍 |

**本手册使用期限**：账户资金 ≤ 30 万期间  
**切换条件**：账户首次达到 30 万，或连续 3 个月回撤超过 20%（说明进攻版不适合当前市场），立即切换回稳健版。

**记住：小资金做大的核心，不是赌，而是“在看得懂的机会上，敢于下注，并严守纪律”。**
