# 项目宪法访问器

## 职责

本 Skill 作为项目编码规范的统一访问入口，供 Agent 在以下场景使用：
- 代码审查时，检查变更是否符合规范
- 编写新代码时，获取命名、类型、错误处理等具体约束
- 回答开发者关于编码风格、架构约定的问题

## 规范来源

所有编码规范定义于：

@.claude/rules/constitution.md

本 Skill 不复制规范内容，仅提供对规范文件的读取与解释能力。

## 使用方式

1. **加载规范**：通过上述 `@` 引用自动加载完整规范内容。
2. **按需提取**：根据调用方的问题或待检查的代码，从规范中定位相关条款。
3. **合规裁决**：将代码与规范进行比对，给出通过/失败结论及具体依据。

## 输出格式

- **查询模式**：返回相关规范条目的原文及必要解释。
- **检查模式**：返回结构化结果：
  - `passed`: true/false
  - `violations`: 违规条目列表，每条包含 `rule`（条款原文）和 `suggestion`（修改建议）

## 约束

- 本 Skill 对规范文件只读，不进行任何修改。
- 若规范文件无法加载，应返回错误并提示：“项目缺少编码规范文件，请创建后重试。”
